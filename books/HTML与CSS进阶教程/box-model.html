border:none和border:0的区别

border:none代表的是border的样式为none，浏览器在解析border:none时不会再渲染，故border:none不占用内存。

border:0表示把border宽度定义成0，虽然在视觉上看不见，但是浏览器依然会对border进行解析和渲染，故border:0会占用内存。
